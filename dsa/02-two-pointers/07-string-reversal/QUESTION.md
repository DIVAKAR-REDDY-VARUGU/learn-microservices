# Reverse String

> **Pattern:** Two Pointers → **String Reversal**  ·  [📖 Learn this pattern](../../README.md#string-reversal)

## Problem

Write a function that reverses a string. The input string is given as an array of characters `s`.

You must do this by modifying the input array **in-place** with `O(1)` extra memory, using two pointers that converge from both ends.

**Constraints:**
- `1 <= s.length <= 10^5`
- `s[i]` is a printable ASCII character.

## Examples

- Input: `s = ['h','e','l','l','o']` -> Output: `['o','l','l','e','h']`
- Input: `s = ['H','a','n','n','a','h']` -> Output: `['h','a','n','n','a','H']`
- Input: `s = ['A']` -> Output: `['A']`

## Your task

Implement the method in **`Answer.java`**, then run `java Answer.java`:

```java
public void reverseString(char[] s) { /* your code */ }
```
